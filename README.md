`mdcat` is a command line tool write in Ruby to display markdown files in the terminal.

## Installation

The recommended way to install `mdcat` is via RubyGems:

    $ gem install mdcat

## Usage

### Displaying a file

    $ mdcat README.md

### For help

    $ mdcat help
